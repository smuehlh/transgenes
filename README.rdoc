== README

A standalone software to enhance mammalian transgenes coupled with a basic web interface.

= Standalone software
Standalone software can be found in lib/standalone subfolder.

= Web interface

_Requirements_
Rails 4.2
Ruby 2.4


_Deployment_
rake assets:precompile # precompile assets
rake db:schema:load # prepare DB
rake autocompletion:update # set up Ensemble autocompletion

rails s # start rails server
